## 匿名函数

```python
# 定义函数
>>> def add(x,y):
...     return x + y
...
>>> # 调用函数
... c = add(3,5)
>>> print c
8


# 使用匿名函数

>>> # 定义函数
... add = lambda x,y : x + y
>>> # 调用函数
... c = add(3,5)
>>> print c
8
```
